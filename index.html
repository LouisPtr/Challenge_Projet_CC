CSS Selectors and Specificity Cheat Sheet

Selector Type Example Description
Element Selectors div Selects all elements of a given type
Class Selector .example Selects all elements with a specific class
ID Selector #example Select an elements with a specific ID
Attribut Selector [type="text"] Selects elements based on their state
Pseudo-class Selector type:hover Selects elements based on their state
Pseudo-element Selector ::first-line Selects a part of an element
